# CLI PROMPT - Paste đoạn này vào Claude Code để bắt đầu build

Đọc file `content-pipeline-prompt.md` trong thư mục hiện tại. Đó là spec chi tiết cho dự án Content Pipeline.

Build dự án theo đúng spec đó. Bắt đầu từ Phase 1 — tạo skeleton project với Express server và toàn bộ frontend UI sử dụng mock data.

Yêu cầu:
1. Tạo đầy đủ cấu trúc thư mục như trong spec
2. Frontend phải hoạt động hoàn chỉnh với mock data (10 bài mẫu tiếng Việt, đa dạng nội dung)
3. Tất cả UI interactions phải hoạt động: expand/collapse, tab gốc/viết lại, loading states, button enable/disable logic
4. Dark theme, responsive, đẹp và chuyên nghiệp
5. Chạy được ngay bằng `npm start`

Sau khi xong Phase 1, tiếp tục Phase 2, 3, 4 theo spec. Mỗi phase hoàn thành thì báo và cho test trước khi sang phase tiếp.
